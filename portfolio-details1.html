서비스 운영 경험 치즈벌룬

  • 프로시저/이벤트를 이용한 DB 데이터 통계 자동화
  • Docker compose를 통한 Ubuntu 서버 환경 구축
  • Docker, Jenkins, Github의 배포 자동화 구축
  • Next.JS, Springboot, FastAPI, ELK 리버스 프록시 세팅
  • MariaDB 기반 데이터 모델링, 구축
  • 간편 JWT 로그인(네이버, 구글, 카카오) 구현
