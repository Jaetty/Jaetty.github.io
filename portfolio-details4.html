Project information

  • 프로젝트: RideWay
  • 주제: 자전거 헬스케어 커뮤니티 서비스
  • 기간: 2023.01 ~ 2023.02
  • 역할: Kotliln AOS 개발, Spring RestfulAPI 개발

  • GitHub: github.com/Jaetty/RideWay

헬스케어 자전거 커뮤니티 제작

  • 주행 경로(GPX) 저장/표시/공유 기능 제작
  • 실시간 운동 정보(칼로리, 시간, 속도) 계산 기능 구현
  • Stomp 라이브러리 기반 실시간 채팅 기능 구현
  • Retrofit2을 이용한 백엔드 서버와의 통신 구현
  • 주행 시 발생하는 Gps 오차 수정(속도, 위치 기반)
  • 회원가입, E-mail 인증, Mypage 수정 기능 구현
  • Service 기반 업적 감지 / 달성 기능 구현

성과 / 배운점
- Kotlin 언어 학습 및 안드로이드 개념 학습
- WebSocket을 이용한 실시간 통신 개념 학습
- 콜백 지옥을 Coroutine 기반 비동기 처리로 해결
- Service를 이용한 백그라운드 개념 학습 및 개발
- 안드로이드에 외부 API (Naver Map, SMS 등) 적용법 학습


교훈
처음으로 Kotlin 안드로이드 개발을 수행했습니다. 시행착오를 겪으며 안드로이드 앱 개발 능력을 갖추게 되었습니다. 특히, 콜백 지옥에 빠질 수 있었는데, Coroutine을 이용하여 개념 및 유지보수성을 높혔습니다. 첫 팀 프로젝트였는데 조장을 수행하며 프로젝트 관리 / 소통의 중요성을 배울 수 있었습니다.
